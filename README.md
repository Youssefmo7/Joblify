# Joblify

A Vue.js application for job seekers, employers, and admins to manage job applications.

## Features

- Candidate Dashboard: View and manage job applications
- Employer Dashboard: Post jobs and review applications
- Admin Dashboard: Manage users and system settings
- Job Listings: Browse available jobs
- User Profiles: Manage personal information

## Project Setup

### Prerequisites
- Node.js
- npm or yarn

### Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm run serve` to start the development server

## Styling
This project uses Tailwind CSS and DaisyUI for styling components.

## User Stories
Based on the provided user stories PDF, this application supports:
- Candidates can register, login, search jobs, apply, and track applications.
- Employers can post jobs, view applicants, and manage their profiles.
- Admins can manage users, jobs, and system configurations.
