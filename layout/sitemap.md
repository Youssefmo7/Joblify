# Sitemap

List of all required pages designed for Joblify, based on the user stories. 
All files are available as pure HTML/CSS in this `layout` folder.

## 1. Authentication
* **File:** `login-register.html`
* **Description:** Unified login and registration page. Users can choose to create an account as a Candidate or an Employer.

## 2. Global / Candidate Views
* **File:** `index.html`
* **Description:** The main job discovery feed. Features a LinkedIn-style center feed with a left sidebar (filters/profile summary) and a top navigation bar. Allows searching by keywords, location, and category.
* **File:** `job-details.html`
* **Description:** Detailed view of a job posting. Includes job requirements, company branding, and an application form (resume upload / contact details).

## 3. Candidate Experience
* **File:** `candidate-dashboard.html`
* **Description:** The candidate's personal dashboard to manage their profile, track application statuses, and see alerts.

## 4. Employer Experience
* **File:** `employer-dashboard.html`
* **Description:** Employer control center. Displays active listings, analytics on applicants, and candidate review queues.
* **File:** `create-job.html`
* **Description:** A form interface for employers to define job titles, descriptions, categories, work types (remote/onsite), and upload company branding.

## 5. Admin Experience
* **File:** `admin-dashboard.html`
* **Description:** System oversight dashboard. Allows admins to approve/reject pending job postings, monitor platform activity, and moderate content.
